this project is my intro to entity-component pattern and most of code here is my try to understand https://simondev.io/ programs

Instructions:

1. mouseover an 'origin' to activate field spawning (this is indicated by a line drawn from origin to mouse pointer)
2. click to spawn a field. If collide with dot of the same color force from field will push dot away from it
3. try to direct a dot to socket to collect score
