this project is an example of entity-component pattern and most of code here is my try to understand https://simondev.io/ programs

Instructions:

1. Mouseover middle-sized dot to activate field spawning (this is indicated by a line drawn from origin to mouse pointer)
2. Click to spawn a field. Field pushes dot away if both are of the same color
3. Try to direct a dot towards socket to score
4. If dot's velocity is less then 3 it is lost. If all dots are lost the game is over
